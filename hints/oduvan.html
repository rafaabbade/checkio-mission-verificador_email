Como posso verificar se um caractere ou string está presente em uma string?

Você pode usar a condição in para verificar se uma substring está presente em uma string. Por exemplo: 

palavra = "exemplo"
if "x" in palavra:
print("A letra 'x' está presente")
else:
print("A letra 'x' não está presente")

Como posso garantir que uma string não comece com um caractere específico?

Você pode usar o método .startswith() combinado com not para verificar se a string não começa com um caractere específico: 

frase = ".começando_com_ponto"
if not frase.startswith("."):
print("A frase não começa com um ponto")
else:
print("A frase começa com um ponto")
