Enunciado

Lucas, que administra um site de eventos, precisa garantir que os emails registrados para notificações sejam válidos para evitar erros de comunicação. Crie uma função que verifique se um endereço de email contém um "@" e ".com". O endereço também não deve começar com @.

Entrada: Uma string (email).

Saída: True se o email for válido, False caso contrário.
